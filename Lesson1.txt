✅ Xulosa:
SQL — bu so‘rov yozish tili.
MySQL va PostgreSQL — bu SQL tilidan foydalanadigan dasturlar (DBMS).
MySQL oddiyroq loyihalar uchun yaxshi.
PostgreSQL kuchli va zamonaviy ilovalar uchun ko‘proq imkoniyat beradi.

💾 DBMS nima?
DBMS (Database Management System) — bu Ma’lumotlar Bazasini Boshqarish 
Tizimi degani. Ya’ni, ma’lumotlarni saqlash, boshqarish, yangilash, izlash 
va o‘chirish imkonini beruvchi dasturiy tizim.

🧩 WebSocket nima?
WebSocket — bu brauzer bilan server o‘rtasida ikki yo‘nalishda (real-time) muloqot qilish imkonini beradigan texnologiya.
🔁 Oddiy HTTP protokol: mijoz (browser) faqat so‘rov yuboradi, server javob beradi.
🔄 WebSocket: doimiy ulanish orqali ikkalasi ham har doim muloqot qila oladi.

🧩 Middleware nima?
Middleware (o‘zbekchasiga "oraliq dastur qatlami") — bu serverdagi so‘rov (request) va 
javob (response) o‘rtasida orasiga qo‘shiladigan kod bo‘lib, u har bir so‘rovni tekshiradi, 
tahrirlaydi yoki yo‘naltiradi.

🧩 JWT (JSON Web Token) 3 ta asosiy qismdan iborat:
Header (Sarlavha)
Payload (Yuk)
Signature (Imzo)

🧩 OSI modeli qatlamlari (7 qatlamli tarmoq modeli)
1. Physical Layer (Jismoniy qatlam)
 Ma’lumotni fiziki tarzda uzatadi (kabellar, signal, bitlar)
2. Data Link Layer (Ma’lumot bog‘lanish qatlam)
 Bitlarni frames’ga ajratadi, xatoliklarni tekshiradi
3. Network Layer (Tarmoq qatlam)
 Ma’lumotni manzillaydi, marshrutlashni amalga oshiradi (IP)
4. Transport Layer (Transport qatlam)
Ma’lumotlarni ishonchli yetkazib beradi (TCP, UDP)
5. Session Layer (Sessiya qatlam)
Aloqani boshqaradi, sessiyalarni o‘rnatadi va tugatadi
6. Presentation Layer (Ta’riflovchi qatlam)
Ma’lumotlarni kodlash/dekodlash, shifrlash
7. Application Layer (Ilova qatlam)
Foydalanuvchiga xizmat ko‘rsatadigan dasturlar, protokollar (HTTP, FTP, SMTP)
